# App Scholar

## Intruções: 

### 1.

Criar database app_scholar no banco de dados

### 2. 
Criar .env na pasta backend usando .env.example como base

### 2. 
Executar o script create_database.sql no Postgres


### 3. 
Expo Go no celular:

No arquivo _frontend/services/api.ts_, substitua localhost pelo IP da sua máquina (ex: http://ipmáquina:3333).

### 4. 
Back
```bash
npm run dev
```

### 5. 
Front
```bash
npm run start 
```
